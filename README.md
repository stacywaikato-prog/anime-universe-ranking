# Anime Universe

A fun anime website I built while learning how to code!  
It shows characters, rankings, hidden gems, and pulls real images from AniList.

This project helped me understand:
- how HTML pages link together
- how JavaScript loads data
- how APIs work
- how to build a real website from scratch
## Features
- Character pages with live images
- Rankings page
- Hidden gems page
- AniList API integration
- Local caching for faster loading

## Technologies Used
- HTML
- CSS
- JavaScript
- AniList GraphQL API
